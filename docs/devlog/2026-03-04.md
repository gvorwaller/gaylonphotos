### Key Technical Insight
- `globalThis.Map` is required when the Svelte component import `Map.svelte` shadows the built-in `Map` constructor — otherwise `new Map()` in `$derived` throws `component_api_invalid_new`
- Google Maps `addListener('click')` works on programmatically-created AdvancedMarkerElements; `addEventListener('gmp-click')` does not fire
- CC3 review caught silent bugs: `photo.thumbUrl` → `photo.thumbnail`, `photo.dateTaken` → `photo.date` — wrong property names produce empty InfoWindows with no error
