### Key Technical Insight
- **Lineage path prefix logic matters**: Replacing "Mother's" with "Madonna's" drops the maternal path segment. The correct transform is to prepend the name and lowercase the existing first letter: `"Mother's X"` → `"Madonna's mother's X"`. For wife lines, simple replacement works since "Wife's" is an extra prefix not part of the genealogical path.
- **Deploy script is authoritative**: Attempting manual `ssh + npm run build` failed due to Cloudflare DNS (domain doesn't resolve to droplet) and host key issues. The deploy script at `./scripts/deploy-to-DO.sh` handles everything correctly.
