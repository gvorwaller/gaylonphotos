### Key Technical Insight
The ancestry `lineagePath` field (e.g., "Father's father's mother") is a lossless encoding of binary tree position. During GEDCOM import, `computeGenerations()` does a BFS walk using actual parent-child links (`familyChild → family.husband/wife`), but only stores the computed path string — the raw links are discarded. For tree visualization, parsing lineagePath back into binary heap indices (`father → pos*2`, `mother → pos*2+1`) reconstructs the full pedigree structure without needing the original GEDCOM relationships.
