### Key Technical Insight
- **HEIC on Linux servers**: sharp's prebuilt binaries don't include HEIC/HEVC decoders (licensing). Building from source requires libvips 8.17.3+ but Ubuntu 24.04 ships 8.15.1. The clean solution is client-side conversion — macOS has native HEIC support via ImageIO, so convert before upload
- **Google Maps API key restrictions**: Keys with HTTP referrer restrictions (for browser use) cannot be used with the Geocoding REST API. Either create a separate unrestricted key or use Nominatim as fallback. Nominatim quality is good for cities/towns, just slower (1 req/sec)
- **PM2 restart kills sessions**: In-memory session Map is cleared on restart. Mid-upload PM2 restart caused 291 auth failures. The retry mechanism handled this cleanly
