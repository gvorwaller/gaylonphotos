### Key Technical Insight
- Svelte 5 `$state(prop.value)` intentionally captures initial value for form editors — the "state_referenced_locally" warning is expected for editable form components (PhotoEditor, ItineraryEditor, GeoTagger). These components make local copies of props that users modify.
- Google Maps wrapper uses vanilla JS + `$effect` instead of svelte-google-maps-api library (which uses Svelte 4 syntax). More control, full Svelte 5 compatibility.
- Photo upload normalizes all formats to JPEG via sharp, using photo ID (not original filename) as the Spaces key.
